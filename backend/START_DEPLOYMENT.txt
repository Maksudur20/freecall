## 🆘 IF SOMETHING BREAKS

### Backend won't deploy?
→ Check Render logs for MongoDB connection error
→ Make sure IP whitelist includes 0.0.0.0/0 in MongoDB Atlas

### Frontend shows 404?
→ Update CORS_ORIGIN on Render backend
→ Make sure backend URL is correct in Vercel env vars
→ Restart both services

### Can't login?
→ Check MongoDB is connected (Render logs)
→ Check JWT_SECRET is set (not blank)
→ Clear browser cache (Ctrl+Shift+Del)

### Files won't upload?
→ Check Cloudinary credentials on Render
→ Check file size (< 5MB for images)
→ Check Cloudinary API key is correct
